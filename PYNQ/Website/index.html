About the Project

This project was developed using a Raspberry Pi and Xilinx's new PYNQ Z-1 board. The Raspberry Pi was developed as a proof of concept and its control page can be found here.

The final system is conprised of a robot, developed using the PYNQ Z-1, which has to modes of operation, a manual mode and an autonomous mode.

Manual mode allows the system's user to control the robot's movements (forward, reverse, left, right, stop) using widgets developed using Python. The manual mode user screen is thr Jupyter Notebook where all of the robot's control code is displayed.

Autonomous mode allows the system to act autonomously. The user selects a lower and upper range of colours in the HSV space using slider widgets. After the values are selected, the image processing algorithm running on the robot will filter out all colour that's outside the specified upper and lower ranges. The algorithm will then turn on the robot's controls so it will follow the desired colour.



Techonology Used During the Project's Development

Raspberry Pi

The proof of concept system was developed on a Raspberry Pi 3 board

Xilinx PYNQ Board

The final system with manual and autonomous mode was developed on Xilinx's now PYNQ Z-1 board.
Available since October 2016.

Python Programming

The robot's control and image processing algorithm were developed using the Python programming language.

Jupyter Notebooks

The Python developemnt was done using Jupyter Notebooks. The Jupyter Notebook server is running on the PYNQ's ARM Cortex-A9 processor.

OpenCV

OpenCV libraries were used for the image processing algorithm.

Linux

The PYNQ's is running a Linux operating system. It's running the 10.15 version of Ubuntu. The operating system allows connectivity to a network, which enable the Internet of Things functionality of the system.
